# dashboard-vendas-powerbi

📊 Dashboard de Vendas no Power BI
Desafio prático da trilha Full-Stack Básico na DIO — criação de um dashboard de vendas com Power BI, replicando e expandindo um relatório a partir de uma amostra de dados disponibilizada pela especialista.

📋 Descrição:
Neste projeto, duas páginas de relatório já criadas durante o curso são replicadas a partir de uma sample de dados. Em seguida, uma terceira página é criada de forma independente, composta pelos seguintes visuais:
🗺️ Mapa 1 — Soma de vendas (sales) e unidades vendidas por país
🗺️ Mapa 2 — Soma de lucro (profit) por país
🍕 Gráfico de pizza — Lucro por segmento

✅ Requisitos do Desafio:
[x] Replicar as duas primeiras páginas do relatório de referência
[ ] Criar a terceira página com os visuais indicados
[ ] Verificar a disposição dos visuais no relatório
[ ] Renomear os visuais para nomes mais claros e diretos
[ ] Ajustar os campos utilizados como dicas de ferramentas (tooltips)
[ ] Publicar o relatório no Power BI Service
[ ] Compartilhar como suplemento no Power Point (ou salvar o .pbix)
